## User Registration & Login

### First Time Users - Registration

#### Step 1: Access Registration
1. On the login page, click **"Register as Parishioner"** button
2. Or click **"New to our parish community?"** link

#### Step 2: Fill Personal Information
**Required Fields:**
- **First Name**: Your given name
- **Last Name**: Your family name
- **Email Address**: Valid email (will be used for login)
- **Password**: Minimum 6 characters
- **Confirm Password**: Must match your password

**Optional Fields:**
- **Date of Birth**: Your birth date
- **Phone Number**: Zimbabwe format (+263...)
- **Address**: Your residential address
- **Emergency Contact Name**: Person to contact in emergencies
- **Emergency Contact Phone**: Their phone number

#### Step 3: Complete Registration
1. Check the **Terms and Conditions** checkbox
2. Click **"Register as Parishioner"**
3. Wait for success message
4. Click **"Sign In Here"** to return to login

### Logging In

#### Quick Login (Demo Accounts)
**For Testing Purposes:**
- **Parishioner Account**: Click "Parishioner Login" button
  - Username: `parishioner`
  - Password: `parish123`
- **Admin Account**: Click "Admin Login" button
  - Username: `admin`
  - Password: `admin123`

#### Manual Login
1. Enter your **Email, Phone, or Username**
2. Enter your **Password**
3. Click **"Sign In"**

#### Login Troubleshooting
- **Invalid credentials**: Check email/username and password
- **Account not found**: Ensure you've registered first
- **Connection error**: Check internet connection and try again

## Troubleshooting

### Common Issues and Solutions

#### Login Problems
**Issue**: "Invalid credentials" error
**Solution**: 
- Check email/username spelling
- Verify password (case-sensitive)
- Try password reset if available
- Contact admin if account locked

**Issue**: "Registration failed" error
**Solution**:
- Check internet connection
- Ensure all required fields filled
- Verify email format is correct
- Try different username if taken

#### Navigation Issues
**Issue**: Page not loading
**Solution**:
- Refresh browser (F5 or Ctrl+R)
- Clear browser cache
- Check internet connection
- Try different browser

**Issue**: Language not switching
**Solution**:
- Click language selector again
- Refresh page
- Clear browser cookies
- Check if JavaScript enabled

#### Admin Dashboard Problems
**Issue**: Cannot access admin features
**Solution**:
- Verify admin login credentials
- Check user role permissions
- Contact system administrator
- Try logging out and back in

**Issue**: User management not working
**Solution**:
- Check database connection
- Verify admin permissions
- Refresh dashboard page
- Contact technical support
